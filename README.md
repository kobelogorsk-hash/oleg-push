# 🔍 Поиск лучших цен на маркетплейсах

Сайт для поиска товаров на Ozon, Avito и Wildberries с учётом репутации продавцов.

## 🚀 Развёртывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы в репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_НИК/repo-name.git
   git push -u origin main
   ```
3. В настройках репозитория перейдите в **Settings → Pages**
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`
5. Нажмите **Save**
6. Через несколько минут сайт будет доступен по адресу: `https://ВАШ_НИК.github.io/repo-name/`

## 📁 Структура проекта

- `index.html` — главная страница
- `styles.css` — стили сайта
- `app.js` — логика поиска товаров

## ⚙️ Как это работает

1. Пользователь вводит название товара
2. Скрипт ищет товары на трёх маркетплейсах (Ozon, Avito, Wildberries)
3. Анализирует репутацию продавцов (рейтинг, отзывы)
4. Сортирует результаты по цене и репутации
5. Показывает 3 лучших варианта с фотографиями и ссылками

## 🔧 Настройка API

В текущей версии используются демонстрационные данные. Для работы с реальными данными:

1. Создайте backend-сервер для парсинга маркетплейсов
2. Обновите функцию `searchProducts()` в `app.js`
3. Замените моковые данные на вызов вашего API
